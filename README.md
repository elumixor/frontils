Utilities for front-end development.

## Extensions Moved

Array, Set, and String extensions have been moved to a separate package: [@elumixor/extensions](https://github.com/elumixor/extensions)

Install it separately if you need those extensions:
```bash
npm install @elumixor/extensions
```
